# Briefing
유튜브 영상 토픽 제공 및 악성 댓글을 필터링 해주는 애플리케이션

## 구성도

## 주요 기능
- 영상 요약 : 키워드를 추출하고 핵심 문장 요약
- 댓글 필터링 : 악성 댓글 및 외국어 댓글 분류
- 데이터 시각화 : 전체 댓글에서 악성 댓글의 비율을 보여줌
- 히스토리 : 사용자가 검색한 영상의 요약본 저장 및 관리

## 적용 기술
- 영상 주제 요약 : STT(SPecch-to-Text), TextRank
- 댓글분석 : 크롤링, LSTM, 데이터마이닝
- 안드로이드
- DB 관리
